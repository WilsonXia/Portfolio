Portfolio Page

Design

This website is reformed and refactored from my last portfolio page from 2022, and utilizes a lot more code and structure. I used HTML, CSS, JavaScript, the Bulma Framework, FireBase, and a lot of other resources to build this site. This was a HUGE improvement from my last portfolio page, which makes me proud of this version.

Some of these resources include:

  • Animista, which helped me create CSS animations
  • Swiper, the basis for the unlimited scroll carousel on the home page
  • Font Awesome, where I got my icons from
  • And tons of Google searches!

I started off by designing the pages of my this site on Krita, a free drawing software on the PC. I mainly focused on the coloring of the page, and utilized Coolers to help create a palette I liked.

Here are some images of the prototypes I made:

I struggled a lot on structuring, and styling the website. There were plenty of times where I wanted to use Bulma for a majority of the project, but realized that in order for the website to style the way I wanted it to, I would need to use my own personal css.

Sizing each component and box was tricky, for example, the project cards on the project's page. They wouldn't maintain the same size as each other unless you predetermined the width and height of each box, and since I wanted to make the website more responsive, it was tricky to balance this goal.

Another tricky part of this project was designing how I would utilize Firebase. It was really easy for me to imagine how I would use the database, but actually producing this feature was a lot harder, as I didn't have a plan to go about making a site to take care of how I would manage the projects to add in the database.
